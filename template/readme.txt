примеры создания шаблонов
.xsl  шаблоны
.xml  примеры данных

Если сайт поддерживает 3 языка надо делать справочник перевода минимум на  русском и английском желательно и казахском.





Для проведения платежа Вы Клиента перенаправляете Post запросом на сайт епей.
Чтобы платежная страница отображалась не в дефолтном формате а в Вашем, у нас должен быть Ваш шаблон.
Это не Джумла шаблон и не страница HTML!
XSL шаблон использует XML данные для построения HTML страницы. Пример XML  данных тоже есть в нашем архиве.

Пример работы разных шаблонов можете посмотреть на нашем сайте.
http://3dsecure.kkb.kz/jsp/client/pay.jsp
сначала попробуйте с дефолтным, затем подставьте  в поле template  название другого шабона
например: killbill.xsl

Название шаблона сделайте другим.
Мы его разместим у себя на сервере.



Многоязычность:

Для многоязычности необходимо все сообщения(ключи) разместить в таких тегах
<i18n:text>bee_pay_attention</i18n:text>

Переводы ключей предоставить в таком формате отдельными файлами
<message key="bee_pay_attention">Будьте внимательны. На проведение платежа выделяется 30 минут и 5 попыток</message>

<message key="bee_pay_attention">Қараңыз! Төлем жүргізу үшін 30 минут пен 5 талпыныс беріледі</message>

Если перевод отсутствует в словаре , то отображается только ключ из тега <i18n:text>
У нас сейчас полноценно поддерживается Английский Казахский и Русский

Для поддержки любого другого языка необходимо перевести  еще все сообщения об ошибках.
